=head1 NAME

HP Calculator Parser Module

=head1 DESCRIPTION

Perl Module for the Assembler C<asm2hpc.pl>.

=head1 INSTALLATION

To install this module type the following:

  perl Makefile.PL
  make
  make test
  make install

=head1 DEPENDENCIES

This module requires these modules:

=over

=item *

Exporter

=item *

Parser::MGC

=back
